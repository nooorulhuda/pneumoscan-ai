Dataset Information

  • Source: Kaggle Chest X-Ray (Pneumonia) Dataset
  • Total Images: 5,856 chest X-rays
  • Training Set: 4,447 images (after proper split)
  • Validation Set: 785 images
  • Test Set: 624 images
  • Classes: Normal vs Pneumonia (Binary)
  • Image Size: 224×224 (299×299 for InceptionV3)
  • Augmentation: Flip, brightness, contrast

Technology Stack

Built with modern deep learning and web technologies:

Python 3.10 TensorFlow 2.19 Keras Flask SQLite OpenCV NumPy scikit-learn Grad-CAM Transfer Learning FPDF2 HTML/CSS/JS

Training Platform:

Kaggle GPU P100 16GB VRAM 30 Epochs Max Early Stopping
🤖

Transfer Learning

5 pre-trained ImageNet models fine-tuned on chest X-ray data with last 30 layers unfrozen.

🗺️

Grad-CAM XAI

Gradient-weighted Class Activation Maps highlight exact lung regions influencing each prediction.

📋

PDF Reports

Download detailed diagnostic reports including X-ray, Grad-CAM heatmap, and prediction details.

🗄️

Patient Database

SQLite database stores all patient records, predictions, and timestamps securely.

⚖️

Class Balancing

Class weights applied during training to handle dataset imbalance between Normal and Pneumonia cases.

🔐

Secure Login

Doctor authentication system with hashed passwords and session management via Flask-Login.
